PRFLR - это система аналитики, направленная на скорейшее обнаружения проблемных по производительности мест в работе приложений. Realtime и непосредственно на Production серверах. Предназначена для использования в связке с системой мониторинга.

PRFLR - не мониторит железо, не строит графики, не собирает логи. Он делает одну простую вещь: помогает найти "бутылочные горлышки" в коде прямо в Production! И ориентирован на большие распределенные кластера серверов.
PRFLR включает в себя:

- сервер сбора данных;
- няшный web-based UI для анализа данных;
- библиотеки под PHP (C++, Python, Perl, .NET, Java, ObjectC в работе) для сборки данных из вашего приложения;
- плагин для трансляции данных в системы мониторинга серверов типа Zabbix, Munin и Nagios.
